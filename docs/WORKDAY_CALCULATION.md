# 出勤日計算機能

## 概要

利用開始日と利用期間から、対象期間内の出勤日数（平日数）を算出する。

---

## API入力項目

### リクエストパラメータ

| パラメータ名 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| start_date | string | ○ | 利用開始日（フォーマット: `YYYY/mm/dd`） |
| period_months | int | ○ | 利用期間（`1`, `3`, `6` のいずれか） |

### リクエスト例

```json
{
    "start_date": "2026/04/01",
    "period_months": 1
}
```

---

## 実装メモ

- まずは平日計算機能のみを実装する。
- 土日を除外した日数を算出する。
- 祝日の除外、リモートワーク頻度の考慮は後続の実装で対応する。

---

## 定期券の利用期間計算ロジック（実装済み）

### 実装内容

`CalcService::calculateExpirationDate()` にて、定期券の有効期間を正しく計算するロジックを実装した。

### 計算ロジック

定期券の有効期間は**暦月単位**で計算される。具体的には「**起算日から N ヶ月後の応当日の前日まで**」となる。

#### 計算例

| 起算日 | 期間 | 終了日 | 説明 |
| --- | --- | --- | --- |
| 4/1 | 1ヶ月 | 4/30 | 5/1の前日 |
| 4/15 | 1ヶ月 | 5/14 | 5/15の前日 |
| 1/31 | 1ヶ月 | 2/28 (または 2/29) | 応当日がない場合は月末 |
| 3/31 | 1ヶ月 | 4/30 | 応当日がない場合は月末 |
| 4/1 | 3ヶ月 | 6/30 | 7/1の前日 |
| 4/1 | 6ヶ月 | 9/30 | 10/1の前日 |
